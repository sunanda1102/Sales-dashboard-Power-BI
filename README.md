# 📊 Retail Sales Dashboard – Power BI Project

## 🔷 Project Overview

This project presents an interactive **Retail Sales Dashboard** built using Power BI to analyze sales performance, revenue trends, product performance, and promotional impact.

The dashboard provides business insights that help in decision-making related to sales growth, inventory management, and marketing effectiveness.

---

## 🔷 Key Features

* 📌 **KPI Metrics**

  * Total Sales: ₹2K
  * Total Revenue: ₹9K
  * Total Orders: 5K

* 📈 **Sales Trend Analysis**

  * Monthly sales trend visualization to identify seasonality and performance patterns.

* 🏬 **Store Performance**

  * Revenue comparison across different stores to identify top-performing locations.

* 📦 **Product Analysis**

  * Top 5 products by revenue
  * Top 5 products by stock levels

* 🎯 **Promotion Impact Analysis**

  * Revenue distribution based on different promotion categories (No Promo, Low, Moderate, High, etc.)

* 📅 **Quarterly Order Analysis**

  * Orders distribution across quarters to understand business cycles.

---

## 🔷 Tools & Technologies Used

* Power BI
* Power Query (Data Cleaning & Transformation)
* DAX (Data Analysis Expressions)

---

## 🔷 Dataset Description

The dataset contains retail transaction data with the following fields:

* Order ID (Unique identifier)
* Product ID
* Store ID
* Order Date
* Sales (Quantity)
* Revenue
* Stock Levels
* Price
* Promotion Type & Category
* Delivery Date

---

## 🔷 Key Insights

* 📊 Highest sales observed in March, followed by a declining trend in later months.
* 🏆 Certain products contribute significantly to overall revenue.
* 🏬 One store generates the highest revenue compared to others.
* 🎯 Promotions have a noticeable impact on revenue distribution.
* 📦 Some products have high stock but low sales, indicating potential inventory issues.

---

## 🔷 Business Use Case

This dashboard can be used by:

* Sales Managers to track performance
* Inventory Teams to manage stock
* Marketing Teams to evaluate promotions

---

## 🔷 Conclusion

This project demonstrates the use of Power BI for transforming raw retail data into meaningful insights through interactive visualizations. It highlights key business metrics and supports data-driven decision-making.
